# sfGuardMessagesPlugin

[![StyleCI](https://styleci.io/repos/49583483/shield?style=plastic&branch=master)](https://styleci.io/repos/49583483)

## Instalation

  * Install the plugin:

    ~~~sh
    $ composer require tomasz-rup/sf-guard-messages-plugin
    ~~~
